# Assets para Expo

Coloca aqui los archivos de imagen necesarios:

- `icon.png` - Icono de la app (1024x1024 px)
- `splash.png` - Pantalla de splash (1284x2778 px o similar)
- `adaptive-icon.png` - Icono adaptivo Android (1024x1024 px)
- `favicon.png` - Favicon para web (48x48 px)

## Diseno recomendado

- Fondo: `#1a237e` (indigo oscuro, mismo que app.json)
- Texto/simbolo: "C" o "CURIEL" en blanco
- Estilo: minimalista, profesional

## Generar con herramientas

Puedes usar:
- [Expo Asset Generator](https://docs.expo.dev/develop/user-interface/app-icons/)
- Canva, Figma, o cualquier editor de imagenes
- Iconos de ejemplo: usa un icono de edificio/inspeccion

## Tamanos minimos

| Asset | Tamano minimo | Recomendado |
|-------|--------------|-------------|
| icon.png | 512x512 | 1024x1024 |
| splash.png | 1284x2778 | 1284x2778 |
| adaptive-icon.png | 512x512 | 1024x1024 |
| favicon.png | 48x48 | 192x192 |
